Class: AnimatedGIFLoader

PIXI.gif.AnimatedGIFLoader

Handle the loading of GIF images. Registering this loader plugin will load all .gif images as an ArrayBuffer and transform into an AnimatedGIF object. Use Resource's animation property to access the object.

new PIXI.gif.AnimatedGIFLoader

Example


 import { Loader } from '@pixi/loaders';
 import { AnimatedGIFLoader } from '@pixi/gif';

 Loader.registerPlugin(AnimatedGIFLoader);
